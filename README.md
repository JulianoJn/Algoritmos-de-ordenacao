# Algoritmos-de-ordenação
Um programa que incrementa algoritmos de ordenação afim de avalia-los em certos casos e suas variações.

## Algoritmos Analisados
- Bubble Sort
- Selection Sort
- Insertion Sort
- Quick Sort
- Merge Sort

## Vetor utilizado
Foi utilizado um vetor de 20 números sendo eles:
int array[] = {9, 3, 7, 1, 6, 2, 8, 5, 4, 10, 15, 65, 90, 120, 1001, 45, 87, 21, 39, 50}; 

## Conclusão e considerações
Um pdf com relatório está no repositório, juntamente com a conclusão e um extras, dado a finalidade de ser entregue perante uma avaliação de atividade, a linguaguem utilizada não se preocupa majoriamente em ser formal e tecnica, descrevendo apenas de forma breve o sistema, implementação e resultados.
